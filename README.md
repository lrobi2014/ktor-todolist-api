# Ktor Todo List API 

Demo TodoList API with Ktor and Exposed
- [What you'll need](#what-youll-need)
- [Get Started](#get-started)
- [To Do](#to-do)

## What you'll need
- Java >= 11.0
- Gradle
- Docker Compose

## Get Started
1. Clone this project.
2. Run `docker compose up`
3. Run ```./gradlew run```
